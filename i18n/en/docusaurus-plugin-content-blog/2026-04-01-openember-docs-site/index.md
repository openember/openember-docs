This site is built with [Docusaurus](https://docusaurus.io/) and explains what **OpenEmber** and **EmberLite** are, how the architecture is layered, how to build each repository, and how the **`ember`** CLI fits in.

<!-- truncate -->

Under **Docs** you will find:

- How OpenEmber relates to EmberLite  
- A high-level architecture overview  
- Build instructions for the main tree and EmberLite  
- Common `ember` commands and environment management (`add` / `use` / `list`)  

The ecosystem is evolving quickly—if you spot gaps or want new topics, please open an issue or PR on [openember-docs](https://github.com/openember/openember-docs).
